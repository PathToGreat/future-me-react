We need to fix the base avatar silhouette.

Current problem:
The current SVG avatar reads as a blob/robot even when the user is in peak condition (lean, muscular, athletic). That prevents connection and undermines the “Future Me” concept.

Goal:
Replace the current SVG avatar renderer with a semi-realistic, human-shaped SVG rig that supports body-type variation (lean, athletic, muscular, overweight) and expression cues without messy overlay lines.

Constraints:
- Keep using SVG (do NOT introduce 3D).
- Avoid drawing random line details over a single shape (this caused visual crowding before).
- Do NOT change layout or restructure the Avatar screen UI.
- Photo upload remains optional. The base avatar must look good without photos.
- The new avatar must remain lightweight and performant on mobile web.

3) PRESETS
Create body-type presets that look HUMAN at a glance:
- LeanAthletic
- MuscularAthletic
- AverageFit
- Soft
- Overweight

Each preset should set proportions in a believable way:
- MuscularAthletic: wider shoulders, thicker arms/legs, stronger V-taper
- LeanAthletic: V-taper but less thickness than muscular
- Soft/Overweight: lower taper, thicker midsection, less definition

4) SVG LAYER STRUCTURE (NO CROWDING)
Render layers in this exact order inside ONE SVG:
1. Background glow layer (optional, subtle)
2. Legs base
3. Torso base
4. Arms base
5. Neck base
6. Head base
7. Shading layer (soft gradients only, minimal)
8. Face features (minimal and semi-realistic, not cartoony)
9. Indicators (very subtle, optional)

Important:
- Do NOT add extra DOM elements behind the SVG.
- All details must be inside the same SVG.
- Use SVG clipPath masking so shading and overlays never spill outside the body regions.
- Avoid outlines as the main detail. Use subtle shading and shape to convey form.

5) SIMPLE EXPRESSION + POSTURE (SUBTLE)
FacialTension should affect:
- eyebrow angle or slight brow compression
- mouth curvature (neutral to slightly downturned)
- eye openness (slightly tighter under tension)

PostureLean should affect:
- a slight forward slump for negative values
- a slight upright stance for positive values

Keep changes subtle. No exaggerated cartoon emotions.
