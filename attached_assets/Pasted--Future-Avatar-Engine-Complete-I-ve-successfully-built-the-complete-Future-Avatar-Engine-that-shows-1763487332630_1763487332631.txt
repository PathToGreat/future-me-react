Simple UI switch above avatar: "Current You" ↔ "Future You"

"Coming Soon" state when insufficient history data (needs 2+ days)

4. Dashboard Integration

Metric bars show either current or projected values

How It Works
Projection Logic:

Requires 2+ days of history data to unlock
Uses getMetricTrend() to calculate individual metric slopes
Projects each metric 90 days forward (clamped to 1-5 range)
Recalculates body composition using projected values
Applies same avatar rendering logic with projected metrics
Example:

Current: Activity 3, Nutrition 3, Sleep 3, Stress 3
Trend: Activity improving (+0.5 per week)
Projected (90 days): Activity 3.7, wider body, better posture, greener glow
